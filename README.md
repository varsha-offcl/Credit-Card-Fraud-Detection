# 💳 Credit Card Fraud Detection System

A machine learning-based web app built with **Streamlit** that detects potential credit card fraud in real time.

## 🚀 Features
- Interactive Streamlit dashboard (improved UI using streamlit-extra)
- Machine learning backend (Random Forest)
- Real-time fraud probability analysis
- Visual performance metrics and insights
- Efficient data visualizations
